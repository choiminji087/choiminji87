🖥️ 기본 컴퓨터 용어 10가지

1. CPU

중앙처리장치로 연산과 명령 제어를 담당하는 컴퓨터의 핵심 부품

2. RAM

작업 중인 데이터를 일시적으로 저장하는 휘발성 메모리

3. HDD

자기 디스크를 이용한 저장장치. 용량은 크지만 속도는 느림

4. SSD

빠른 속도의 저장장치로, 하드디스크보다 데이터 접근이 빠름

5. IP 주소

인터넷에 연결된 장치를 식별하는 고유 주소

6. OS (운영체제)

하드웨어와 소프트웨어를 관리하고 사용자와 컴퓨터 사이를 연결

7. URL

웹 상의 자원의 주소를 나타냄 (예: https://example.com)

8. DNS

도메인 이름을 IP 주소로 변환해주는 시스템

9. BIOS

컴퓨터가 부팅될 때 가장 먼저 실행되는 시스템 소프트웨어

10. GPU

그래픽 연산을 전문적으로 수행하는 장치
